Modules
=========

Cette page contient tout les modules qui sont en développement ou déjà conçus.

Les différents modules sont:

- Communication:
--------------
 Permet la communication entre le drone et une arduino, surement développé en plusieurs versions. A cette date (Version 0.1), la communication se fait par radio NRF24.

- Main:
-----------
 Le module Main n'est pas un module à proprement parler car il est essentiel au fonctionnement des autres modules. Il s'occupe du système de log, d'initialiser les modules, ...
